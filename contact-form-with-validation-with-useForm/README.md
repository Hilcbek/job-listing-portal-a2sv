# Contact Form Application

A modern, responsive, and animated contact form built with **React**, **React Hook Form**, **Zod**, **Tailwind CSS**, and **Tabler Icons**. The form includes real-time validation, smooth animations, and toast notifications using **Sonner**.

---

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices.
- **Form Validation**: Built with **React Hook Form** and **Zod**, providing real-time validation for all fields.
- **Animated UI**: Smooth animations for form elements on mount and interaction.
- **Async Submission**: Mock async submission with loading state on the submit button.
- **Toast Notifications**: Success and error notifications using **Sonner**.
- **Professional Styling**: Clean and modern input fields, buttons, and card layout.

---

## Technologies Used

- **React** – Frontend framework
- **React Hook Form** – Form handling and validation
- **Zod** – Schema validation
- **Tailwind CSS** – Utility-first styling
- **Tabler Icons** – Professional icons
- **Sonner** – Toast notifications
- **TypeScript** – Type safety
